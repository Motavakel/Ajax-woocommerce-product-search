== Description ==

Open Source Jalali Date Converter Plugin
This open-source plugin uses PHP and JavaScript libraries to convert Gregorian dates to Jalali (Shamsi) dates
 in WordPress. By using this plugin, users whose date calendar is Jalali, such as Iranians, will no longer
 face challenges related to the calendar and converting Gregorian dates to Jalali in WordPress, and they can easily manage their dates.


== Installation ==

## Automatic Installation:
    Log in to your WordPress dashboard.
    Navigate to the "Plugins" menu and click "Add New."
    In the search box, type "Jalali Date Picker."
    Find the plugin, then install and activate it.

Manual Installation:
    Download the plugin as a zip file.
    Go to your WordPress dashboard, navigate to the "Plugins" menu, and click "Add New."
    At the top of the page, click "Upload Plugin."
    Upload the plugin's zip file, then activate it.

## After activation, all settings will be applied automatically.

== Frequently Asked Questions ==

= In which sections does the date conversion occur using this plugin? =
The date conversion occurs in posts, pages, and all necessary areas of WooCommerce.

Yes, it is fully compatible with WooCommerce
